Skyracer - The Return

Description:
Sky Racer is a fast-paced 3D endless runner game inspired by classics like Temple Run. Built in Unity with C#, the game challenges players to dodge obstacles, collect coins, and survive as long as possible.

Gameplay Features:

  • Dynamic Obstacles: Randomly generated obstacles of varying sizes and positions keep each run unpredictable.

  • Coin Collection: Players earn points by collecting coins scattered along the road.

  • Life System: Players start with 3 hearts. Each collision with an obstacle costs 1 heart, and on the fourth hit, the game ends.

  • Progressive Speed: The player’s speed increases after collecting a certain number of coins, up to a maximum speed limit, raising the challenge as the game goes on.

  • Game Over Mechanic: When all lives are lost, the run ends, encouraging replayability.

Tech Used:

  • Unity Game Engine

  • C# Scripting

  • Basic 3D Modeling & Physics

  • UI for score, coins, and life system

Status:
Completed and published as a standalone build (installer available).

LeHydroSys - Lettuce hydroponic system monitoring

Description:
LeHydroSys is a complete IoT-based hydroponic farming system designed for lettuce cultivation. It integrates real-time monitoring with automated control to create a smart and efficient growing environment.

Features:

  • IoT Monitoring: Tracks water level, temperature, and humidity through sensors.

  • Automatic Water Circulation: Ensures continuous water flow to keep the hydroponic system healthy.

  • Automatic Sprinkler Control: Activates when temperature rises above a threshold, protecting lettuce growth from heat stress.

  • Web Dashboard: Provides real-time monitoring, logs, and data visualization.

  • Android App: Offers portable monitoring and alerts, giving farmers instant access to their system.

  • Scalable Design: Can be extended with additional sensors or automation features.

Tech Stack:

  • IoT Devices: ESP32/Arduino with water  & air temperature sensors, humidity, pH, salinity and water level sensors (c++)

  • Backend: Node.js webserver

  • Mobile: Android app (Java)

  • Actuators: Pump for water circulation, sprinkler system for temperature control

Status:
Completed as a capstone project, integrating hardware, software, and mobile applications into a unified IoT solution.
